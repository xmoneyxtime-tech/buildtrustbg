# Business Vision

## Mission

BuildTrustBG ще бъде най-доверената платформа за строителни фирми в България.

Основната цел не е просто списък с фирми, а изграждане на доверие между клиентите и изпълнителите.

---

## Revenue Model

Платформата ще се издържа чрез:

- Premium профили
- Featured фирми
- Gold Verification
- Банер реклама
- Платени запитвания (в бъдеще)
- Допълнителни услуги за фирмите

---

## Core Principles

- Trust First
- Real Companies Only
- Quality Over Quantity
- Long-term Customer Relationships
- Transparency
- Honest Reviews
- Verified Companies

---

## Long-Term Goal

BuildTrustBG трябва да се превърне в най-разпознаваемата строителна платформа в България.

След това платформата може да се разшири в други държави от Балканите.

---

## Marketplace Vision

BuildTrustBG не е фирмен сайт.

BuildTrustBG е Marketplace.

Платформата свързва:

- Клиенти
- Строителни фирми
- Архитекти
- Проектанти
- Интериорни дизайнери
- Инженери
- Доставчици

---

## Success Metrics

Успехът се измерва чрез:

- брой реални фирми
- брой проверени фирми
- брой успешни запитвания
- доволни клиенти
- месечни Premium абонаменти
- доверие към платформата
---

# Development Rules

Every new feature must follow these principles:

- Reusable components
- Responsive design
- Accessibility
- SEO optimization
- No fake content
- No fake companies
- No fake reviews
- Clean TypeScript
- Modular architecture
- Production-ready code

Before every release:

- Run npm run lint
- Run npm run build
- Fix every warning
- Commit
- Push to GitHub
- Verify deployment on Vercel
